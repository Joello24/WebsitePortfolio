Universal Directory Visualizer

This project was developed by myself and my three team members in our CS458 senior capstone class. We built the application with as an electron app consisting of a java backend connecting to angular/nodejs frontend.

We built the application with as an electron app consisting of a java backend connecting to angular/nodejs frontend.

The application prompts the user to select a file system or drive in their local storage, recursively storing the nested folders and files into hashmaps which are sent to the frontend to be displayed. The idea was to display the files as a series of blocks dependant on the size of file, making it easy for the user to select a box and see what is taking up space in specific folders/drives.

Aim World

This game was developed by myself in UE4. It's an arcade shooter to help improve your aim by tracking your scores. Key features include a custom FPS weapon and coresponding animations and sounds, multiple levels of difficulty and target selection which are all modifyable through the custom shootable menu, and health/ammo/score tracking built into the HUD.
